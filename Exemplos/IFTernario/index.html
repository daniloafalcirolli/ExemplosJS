If aplicado dentro de uma função que retorna um um valor booleano, true ou false. ( Imagem 1 )

Esse é o if padrão do JavaScript nenhuma adição, é o que vocês já conhecem do Java, descomente da linha 4 até a 12 para ver o resultado no console do seu navegador.

Ternario aplicado da mesma forma que o exemplo anterior, comparando se 2 numeros são iguais, true ou false. ( Imagem 2 )

Ternario tem o intuito de diminuir o if e deixar o codigo menor, podendo ser aplicado em lugares completamente inusitados, deixando sua usabilidade maior, descomente das linhas 15 a 17 para ver os resultados no console.
