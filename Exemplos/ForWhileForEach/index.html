For, While e ForEach

While

Sendo uma das formas menos recomendada de rodar um vetor/array ou qualquer coisa que precise passar por uma lista de informações, mas em contra partida uma das melhores formas de fazer um loop com condições especificas.

Rodando uma array com while ( Forma menos eficiente, Imagem 1 )

Com essa forma de se passar pelas informações de um array/vetor sendo a menos utilizada e a que mais consome linhas de codigo eu lhes peço , não use isso em seu codigo, while tem aplicações melhores que isso.
Mas não no JavaScript Hihi, esse exemplo quando for executado no seu console do navegador vai dar o seguinte resultado: "Giulia", "José" e "Zézinho", nessa respectiva ordem. O codigo está a ser descomentado está dentre as linhas 5 e 10, apenas descomente e abra seu console que estará lá o resultado da leitura do array.

For

Sendo uma das minhas formas favoritas de passar por informações de um vetor ainda é necessario escrever muito para chegar a forma de passar pelas informações

Rodando um vetor/array com for, não é a forma que eu mais gosto por ter que escrever muito, mas é bem eficiente e fica melhor que o while dentro do codigo ( Imagem 1 )

Para min é a forma mais complicada de se passar por um array/vetor com for, então não recomendo muito, mas por já estar estabelecido o conceito por conta do Java não é algo que seja necessario alterar se não quiser aprender algo diferente

Rodando um vetor/array com for, agora mudando sua estrutura para um for in sem mostrar o item do array, apenas o seu respectivo numero ( Imagem 2.1 )

For in é um estrutura mais simples que remove a condição, ele apenas passa para a variavel "n" o numero do item do array, então caso abra se console e descomente das linhas 21 até 24 percebera que ira aparecer os numero em 0, 1 e 2, em order pois é os numeros respectivos de cada item do array em ordem.
